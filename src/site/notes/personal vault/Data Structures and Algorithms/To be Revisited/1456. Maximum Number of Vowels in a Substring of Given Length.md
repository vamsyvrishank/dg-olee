Given a string `s` and an integer `k`, return _the maximum number of vowel letters in any substring of_ `s` _with length_ `k`.

**Vowel letters** in English are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

**Example 1:**

**Input:** s = "abciiidef", k = 3
**Output:** 3
**Explanation:** The substring "iii" contains 3 vowel letters.

**Example 2:**

**Input:** s = "aeiou", k = 2
**Output:** 2
**Explanation:** Any substring of length 2 contains 2 vowels.

**Example 3:**

**Input:** s = "leetcode", k = 3
**Output:** 2
**Explanation:** "lee", "eet" and "ode" contain 2 vowels.

**Constraints:**

- `1 <= s.length <= 105`
- `s` consists of lowercase English letters.
- `1 <= k <= s.length`

```cpp

class Solution {
public:
    int maxVowels(string s, int k) {

        set<char> st = {'a','e','i','o','u'};

        int start = 0;
        int end = 0;
        int temp = 0;
        int ans = 0;

        while(end < s.size()){
            if(st.contains(s[end])){
                temp++;
            }
            if(end-start+1 ==k){
                ans = max(ans , temp);

                //now decrement if s[start] contains vowel.
                if(st.contains(s[start])){
                    temp--;
                }
                start++;
            }
            end++;
        }
        return ans;


    }
};
```

```python 

class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        start = 0
        end = 0
        count = 0
        ans = 0
        vowels = {'a', 'e', 'i', 'o', 'u'} 

        while end < len(s):
            # If the new character is a vowel, increase our current count
            if s[end] in vowels:
                count += 1
            
            # When the window hits exactly size k
            if end - start + 1 == k:
                # Record the max vowels found so far
                ans = max(ans, count)

                # Shrink the window: if the character leaving is a vowel, decrease count
                if s[start] in vowels:
                    count -= 1
                
                start += 1
    
            end += 1
        return ans
```
