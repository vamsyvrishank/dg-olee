## 1. **Two Pointers**

- Same direction, opposite direction, fast-slow
- _Examples:_ Remove duplicates, container with most water, 3sum

## 2. **Sliding Window**

- Fixed size, variable size, substring problems
- _Examples:_ Longest substring without repeating characters, minimum window substring

## 3. **Binary Search**

- Classic search, search in rotated array, find boundaries
- Search on answer space (monotonic conditions)
- _Examples:_ Find peak element, search in 2D matrix, aggressive cows

## 4. **Tree Traversals**

- DFS (inorder, preorder, postorder), BFS, Morris traversal
- _Examples:_ Validate BST, LCA, serialize/deserialize

## 5. **Graph Algorithms**

- DFS, BFS, topological sort, cycle detection
- Dijkstra, Bellman-Ford, Floyd-Warshall, Union-Find
- _Examples:_ Course schedule, number of islands, clone graph

## 6. **Dynamic Programming**

- 1D DP, 2D DP, knapsack variants, LCS/LIS
- _Examples:_ Coin change, edit distance, house robber, partition problems

## 7. **Backtracking**

- Permutations, combinations, subsets
- Constraint satisfaction problems
- _Examples:_ N-Queens, Sudoku solver, word search

## 8. **Greedy Algorithms**

- Interval scheduling, activity selection
- _Examples:_ Jump game, meeting rooms, task scheduler

## 9. **Stack/Queue**

- Monotonic stack, monotonic queue
- _Examples:_ Next greater element, largest rectangle in histogram, sliding window maximum

## 10. **Heap (Priority Queue)**

- K-way merge, top K elements, running median
- _Examples:_ Merge K sorted lists, find median from data stream

## 11. **Linked List**

- Reversal, cycle detection (Floyd's), merge operations
- _Examples:_ Reverse nodes in k-group, intersection of two linked lists

## 12. **Hash Map/Set**

- Frequency counting, seen/unseen tracking
- _Examples:_ Two sum, group anagrams, LRU cache

## 13. **Bit Manipulation**

- XOR tricks, bit masks, subset generation
- _Examples:_ Single number, counting bits, subsets

## 14. **Trie**

- Prefix matching, autocomplete
- _Examples:_ Implement trie, word search II

## 15. **Intervals**

- Merge, insert, intersection
- _Examples:_ Merge intervals, meeting rooms II

## **Interview Distribution (Rough)**

- Arrays/Strings + Two Pointers/Sliding Window: **30%**
- Trees/Graphs: **25%**
- Dynamic Programming: **15%**
- Stack/Queue/Heap: **10%**
- Backtracking/Recursion: **10%**
- Others: **10%**
